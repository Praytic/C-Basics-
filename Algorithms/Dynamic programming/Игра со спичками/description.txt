Двое играют в следующую игру. Имеется M спичек. За один ход можно взять a[1], либо a[2], ... либо a[n] спичек. Проигрывает тот, кто не может сделать ход. Ваша задача определить, кто выиграет при правильной игре обоих. Если это первый игрок, то выведите в первой строке FIRST, иначе SECOND. Если первый игрок победит, то во второй строке выведите такое i, что взяв a[i] спичек на первом ходу первый игрок гарантирует себе победу при правильной дальнейшей игре.

Входные данные
В первой строке задано два натуральных числа N (1 <= N <= 10) и M (1 <= M <= 106). Во второй строке содержится N натуральных чисел - a[1], a[2], ..., a[N], не превосходящих 20.

Выходные данные
Выведите ответ в указанном выше виде. Если выйгрышный ход определн не однозначно, выведите любой.

Пример

Ввод
Пример 1 
3 5 
3 2 1 

Пример 2 
1 3 
2

Вывод
Пример 1 
FIRST 
3 

Пример 2 
FIRST 
1
