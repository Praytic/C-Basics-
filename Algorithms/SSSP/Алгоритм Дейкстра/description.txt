Задан взвешенный неориентированный граф из N вершин. Требуется найти длину кратчайшего пути из вершины v1 в вершину v2.

Входные данные
В первой строке через пробел записано четыре натуральных числа N, M, v1, v2 (1 <= N <= 500, 1 <= M <= N*N, 1 <= v1,v2 <= N). 
Каждая из следующих M строк содержит описание одного ребра графа: три числа - начало, конец и длина. Длина ребра - неотрицательное целое число, и не превосходит 1000. 
Никакое ребро во входном файле не встречается дважды.

Выходные данные
В выходной файл вывести единственное число - длину кратчайшего пути из вершины v1 в вершину v2, или -1 если такого пути не существует.

Пример

Ввод
2 1 1 2 
2 1 2

Вывод
2
