Дан ориентированный граф. С ним проводят две операции: 
1)	добавляют дугу из вершины v1 в вершину v2 длины 1 или 2; 
2)	запрашивают длину кратчайшего пути из вершины v1 в вершину v2. 
В начальный момент времени данный орграф пустой (то есть не содержит ни одной дуги). Ваша задача для каждой операции второго типа выдать искомое число или -1, если пути из v1 в вершину v2 не существует. 
Между данной парой вершин может быть более одной дуги.

Входные данные
В первой строке записано число N (1 <= N <= 40) - количество вершин в орграфе и число M (1 <= M <= 50000) - количество проводимых операций. Описание операции первого типа имеет вид "1 v1 v2 L" (L - длина дуги, 1 <= L <= 2), операции второго типа - "2 v1 v2".

Выходные данные
Для каждой операции второго типа выведите искомое число в отдельную строку.

Пример

Ввод
4 3 
2 1 2 
1 1 2 1 
2 1 2

Вывод
-1 
1
