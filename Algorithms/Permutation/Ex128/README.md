### 128. Подсчет числа циклов в перестановке

<pre>ограничение времени на тест: 0.25 сек.
ограничение памяти на тест: 65536 KB.</pre>

Дана перестановка. Найдите число циклов в ней.

**Входные данные**

Число N (1 <= N <= 10000), затем перестановка.

**Выходные данные**

Выведите ответ.

**Пример**

<pre>Ввод
5 
1 4 3 5 2

Вывод
3</pre>
