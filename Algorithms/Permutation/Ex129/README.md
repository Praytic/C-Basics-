### 129. Подсчет наименьшего числа обменов

<pre>ограничение времени на тест: 0.25 сек.
ограничение памяти на тест: 65536 KB.</pre>

Дана перестановка. Наименьшее число обменов, чтобы ее отсортировать.

**Входные данные**

Число N (1 <= N <= 10000), затем перестановка.

**Выходные данные**

Выведите ответ.

**Пример**

<pre>Ввод
5 
1 4 3 5 2

Вывод
2</pre>
