### 145. Номер перестановки

<pre>ограничение времени на тест: 0.25 сек.
ограничение памяти на тест: 65536 KB.</pre>

Дана перестановка. Найдите N - количество элементов в ней и K - ее номер в лексикографическом упорядочивании.

**Входные данные**

Задана перестановка. Количество элементов в ней не больше 20.

**Выходные данные**

Выведите N и K. Гарантируется, что найденное K <= 2 * 10<sup>9</sup>.

**Пример**

<pre>Ввод
1 3 2

Вывод
3 2</pre>
