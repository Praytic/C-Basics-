### 3.7 Упражнение III

###### Ex 20

Вывести на экран: Все трехзначные числа, в которых хотя бы две цифры повторяются.

**Замечание**

Решите каждую задачу тремя способами: использую операторы цикла while, do while и for.
В циклах while решить задачу в определенных границах, заданных пользователем.

**Пример(ы)**

<pre>Ввод
200 200

Вывод
200</pre>

<pre>Ввод
-50 110

Вывод
100 101 110</pre>
